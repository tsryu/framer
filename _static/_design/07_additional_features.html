이 섹션에서는 프레이머 디자인에서 사용할 수 있는 모든 추가 도구에 관해 설명합니다. 레이어 목록, 캔버스 또는 캔버스 자체에서 레이어를 마우스 오른쪽 클릭하여 여러 가지 기능을 사용할 수 있습니다.

앞, 뒤로 이동 (Forward & Backward)

이를 사용하여 계층 구조의 재정렬을 빠르게 수행할 수 있습니다. 레이어를 선택하고 Forward를 클릭하면 레이어 목록에서 레이어가 위로 이동하고 Backward을 클릭하면 레이어가 아래로 이동합니다. 레이어를 앞뒤로 이동하면 자식 레이어도 함께 이동합니다.

복제 (Duplicate)

레이어 또는 아트보드를 선택하고 'Duplicate'를 마우스 오른쪽 버튼으로 클릭하면 캔버스에 포함된 모든 속성 및 자식 레이어와 함께 캔버스에 복사하여 붙여 넣습니다. 중복된 아트보드가 원래 레이어 위에 겹쳐지는 반면, 아트보드는 원본 아트보드 오른쪽에 중복됩니다.

삭제 (Delete)

레이어 또는 아트보드를 선택하고 'Delete'를 마우스 오른쪽 버튼으로 클릭하면 해당 레이어에 포함된 모든 자식 레이어와 함께 즉시 제거됩니다.

계층 구조에서 삭제 (Delete from Hierarchy)

모든 자식 레이어를 유지하면서 부모 레이어만 삭제하려면 상위 레이어를 선택하고 마우스 오른쪽 버튼을 클릭한 다음 'Delete from hierarchy'를 선택하십시오. 이렇게 하면 부모 레이어가 제거되고 아트보드 또는 캔버스의 원래 계층 구조를 반환됩니다.

부모 추가 (Add Parent)

둘 이상의 레이어를 선택하고 '‘Add parent'를 마우스 오른쪽 버튼으로 클릭하면 두 레이어가 그룹이 되어 보이지 않는 부모 레이어 아래에 중첩됩니다. 자동 반응형 레이아웃 규칙(automatic responsive layout rules)을 사용하여 두 레이어가 이제 한 개의 요소로 작동합니다. 부모 크기를 조정해도 자식 레이어에게 영향을 미치지 않습니다.
