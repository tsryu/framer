크기, 반지름, 채우기, 위치, 그림자 및 오퍼시티 등 레이어의 스타일 요소를 지정합니다. 정렬 도구와 라이브 시뮬레이션을 사용하여 레이어를 잠그거나 잠금 해제하세요.

정렬 속성

디자인 캔버스의 오른쪽에는 속성 패널이 있습니다. 다른 그래픽 툴과 마찬가지로 특정 레이어의 속성을 지정하는데 필요한 모든 도구가 있습니다. 패널 상단에는 정렬 도구 옵션이 있습니다. 그것들을 사용하여 레이어를 아트보드 그리고 다른 레이어를 기준으로 스마트하게 배치합니다.

아트보드 속성

채우기 색상, 크기 또는 위치 변경과 같이 아트보드의 속성을 변경할 수 있습니다. 아트보드 반응형 속성을 테스트하려면 한 번에 하나 이상의 속성을 선택한 다음 기기를 전환하세요. 만약 여러분의 콘텐츠가 제대로 정렬되었다면, 자동으로 아트보드의 크기에 맞춰 변형될 것입니다.

레이어 속성

레이어의 속성을 보려면 레이어 목록에서 하나 이상의 레이어를 선택하거나 캔버스에서 직접 선택할 수 있습니다. 프레이머는 각 모서리의 반경(border-radius)과 같은 추가속성을 포함하여 거의 모든 속성을 제공합니다.

프레이머 디자인에서는 레이어의 크기와 위치를 가장 세분화하여 제어할 수 있는 기능을 제공합니다. 다음 섹션에서는 잘 사용하면 상상 이상으로 강력한 기능이 될 수 있는 고급 기능에 대해 자세히 설명합니다.

레이어 위치 지정

새 레이어(자식)를 다른 레이어(부모) 위에 배치하면 프레이머는 위치를 인식하고 부모 레이어 및 아트보드에 연결합니다(anchoring). 이것은 디자인 작업방식의 효율성을 크게 증가시킵니다. 기준점(anchor point)을 변경해야 하는 경우, 속성 패널의 위치 지정 도구를 사용하여 수동으로 설정할 수 있습니다.

예를 들어, 메뉴바 레이어(자식)를 탭 바 레이어(부모)의 오른쪽에 배치하면 프레이머는 메뉴바가 탭 바에 대해 고정된 위치에 자동으로 맞춰지도록 설정됩니다. 속성 패널을 사용하여 언제든지 레이어의 기준점을 변경할 수 있습니다. 프레이머 디자인은 여러분이 작업하기 쉽게 기준점 변경을 실시간 시뮬레이션으로 확인할 수 있습니다. 시뮬레이션은 선택한 제약 조건을 기반으로 레이어의 반응형 성질을 보여줍니다.

레이어 크기 조정

워크플로우의 효율성을 높이기 위해 프레이머는 부모 레이어(아트 보드 포함)의 크기를 조정하는 방법을 모방하여 모든 하위 레이어의 크기를 자동으로 조정합니다.

레이어 클리핑

프레이머 레이어는 레이어 그룹보다는 부모 레이어 - 자식 레이어 관계 내에서 작동하기 때문에 클리핑과 마스킹이 다르게 처리됩니다. 레이어를 마스킹하려면 부모 레이어 위에 자식 레이어를 놓고 속성 패널에서 클립을 클릭합니다. 그러면 자식 레이어가 부모 레이어의 경계로 고정됩니다.

텍스트 정렬 속성

프레이머에서 텍스트 레이어는 스마트 속성을 가진 레이어입니다. 텍스트 레이어는 부모 레이어(또는 아트 보드)에 배치되는 위치에 따라 논리적으로 정렬됩니다.

예를 들어, 탐색바의 왼쪽에 배치된 텍스트 레이어는 부모 레이어나 아트보드의 크기가 조정되거나, 크기가 조정되는 방식에 상관없이 즉시 왼쪽 정렬 및 기준점이 됩니다. 텍스트 레이어를 선택하고 속성 패널에서 위치 지정 도구를 사용하여 언제든지 자동 정렬의 제약 조건을 변경할 수 있습니다.

프레이머에서 텍스트 크기 조정은 텍스트 크기와 레이어 자체를 모두 변경할 수 있습니다. 텍스트 필드의 크기를 조정하면 레이어 제한 영역 크기도 변경되어, 텍스트 블록의 너비를 변경하는데 최적입니다.
