프레이머로 디자인 한 레이어를 코드 탭에서 사용하려면 타겟팅을 사용해야 합니다. 필요한 레이어만 타겟팅하면 코드 깔끔하고 작업하기가 훨씬 쉬워집니다.

타겟팅이 활성화된 레이어만 코드에서 타겟팅 할 수 있으며 코드 탭의 레이어 목록에 표시됩니다.

타겟팅을 사용하려면 레이어 목록의 레이어 오른쪽에 있는 타겟 아이콘을 클릭하거나 'Rename to Target'를 마우스 오른쪽 버튼으로 클릭하세요. 레이어에서 타겟팅을 사용하면 입력 필드가 활성화됩니다. 만약 여러분이 레이어의 이름을 바꿨다면, 프레이머가 이름을 자동으로 프레이머 코드 에디터에서 읽을 수 있는 형식으로 수정할 것입니다. 레이어 이름을 변경하지 않은 경우에는 이름을 삽입하라는 메시지가 표시됩니다. 레이어 오른쪽에있는 미리 보기는 레이어가 코드 탭에서 호출되는 것을 보여줍니다.

타깃 레이어에는 이름 옆에 활성화된 타깃 아이콘이 있습니다. 타겟을 삭제하거나 이름을 변경하려면 레이어 목록에서 해당 레이어를 마우스 오른쪽 버튼으로 클릭하고 'Delete Target'또는 'Rename Target'을 선택하세요.
