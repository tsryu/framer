배경 레이어는 일반 레이어지만 창에 맞게 크기가 조정되어 항상 전체 캔버스를 포함합니다. 배경 레이어에 부모 레이어가 있는 경우, 부모 레이어 크기를 상속합니다.


layerA = new BackgroundLayer
    backgroundColor: "white"

색상은 키워드(ex. black, white 등등)뿐만 아니라 RGB 및 16 진수(Hex)값으로 정의할 수 있습니다.


# 흰색 16진수 값
layerA = new BackgroundLayer
    backgroundColor: "#ffffff"
 
# 흰색 RGB 값
layerA = new BackgroundLayer
    backgroundColor: "rgb(255,255,255)"
 
# 흰색 RGBA 값 (RGBA는 RGB에서 투명도를 추가한 색상 단위입니다)
layerA = new BackgroundLayer
    backgroundColor: "rgba(255,255,255,1)"
