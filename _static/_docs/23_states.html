레이어 상태는 다양한 레이어 속성의 조합을 관리하고 구성합니다. 애니메이션은 상태를 지정해서 사용할 수 있습니다. 스위치를 사용하면 즉시 다른 상태로 이동할 수 있으며, 순환(cycle)으로 상태에서 상태로 순차적으로 이동할 수 있습니다.

상태에 x 및 y와 같은 레이어 속성을 추가할 수 있습니다. html과 같이 애니메이션 가능하지 않은 레이어 속성은 상태 애니메이션의 끝에 적용됩니다.

상태는 애니메이션 옵션 속성을 가질 수 있습니다. 레이어가 정의된 상태로 애니메이션이 적용될 때마다 사용됩니다.

아래 세 가지 사전 정의된 상태는 이미 용도가 지정되어 있어서 사용할 수 없습니다. 여러분은 current(현재) 및 previous(이전)의 name 속성을 사용하여 원하는 상태를 가져올 수 있습니다.

  1. default — 초기 상태
  2. current — 현재 활성 상태
  3. previous — 이전 활성 상태

예시: 상태 생성 및 삭제


# 상태 생성
layerA.states.stateA =
    x: 100
 
# 상태 삭제
delete layerA.states.stateA

예시: 애니메이션 옵션을 가지고 있는 상태


# animationOptions으로 상태 생성
layerA.states.stateA =
    x: 100
    animationOptions:
        curve: Spring(damping: 0.5)
        time: 0.5

예시: 상태로 애니메이트


# 상태 생성
layerA.states.stateA =
    x: 100
 
# 상태로 애니메이트
layerA.animate("stateA")

예시: 상태로 스위치 하기


# 상태 생성
layerA.states.stateA =
    x: 100
 
# 애니메이션 없이 상태 스위치하기
layerA.stateSwitch("stateA")

예시: 상태 순환하기


# 한 번에 여러 상태 정의하기
layerA.states =
    stateA:
        x: 100
    stateB:
        x: 200
 
# 클릭으로 이전, 다음 상태로 이동하기
layerA.onTap ->
    layerA.stateCycle("stateA", "stateB")

layer.states.current <object>

현재 활성 상태의 객체를 반환합니다.

속성

  1. name — 상태 이름 객체를 반환하는 읽기 전용 속성입니다.

layerA = new Layer
 
layerA.states.stateA =
    x: 500
    opacity: 0.5
 
layerA.stateSwitch("stateA")
 
print layerA.states.current
# 결과: 상태 객체
 
print layerA.states.current.name
# 결과: stateA

layer.states.previous <object>

이전 활성 상태의 객체를 반환합니다.

속성

  1. name — 상태 이름 객체를 반환하는 읽기 전용 속성입니다.

layerA = new Layer
 
layerA.states.stateA =
    x: 500
    opacity: 0.5
 
layerA.stateSwitch("stateA")
 
print layerA.states.previous
# 결과: 상태 객체
