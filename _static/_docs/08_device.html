DeviceComponent는 iPhone, iPad 또는 Android와 같은 기기를 에뮬레이션합니다. 기기와 내용의 크기를 변경하고 방향(가로 보기, 세로 보기)을 조정할 수 있습니다. 기기를 설정이 끝나면 화면에 렌더링됩니다. 프레이머 미리보기는 미러 기능을 사용한 미리보기와 일치합니다. 즉, 프레이머 프리뷰로 아이폰을 보고 있다면, 미러 기능으로 실제 아이폰에서 봤을 때는 전체 화면으로 똑같이 나타납니다.

크기변경 및 기기방향 속성에는 Framer.Device 접두사가 붙습니다.


Framer.Device.contentScale = 0.5
Framer.Device.orientation = 90

기기는 모음으로 이해하면 쉽습니다. 따라서 쉽게 속성을 추가하거나 커스터마이즈 할 수 있습니다. Github에서 전체 소스를 찾을 수 있는데요. 다음은 몇 가지 일반적인 예입니다.


# 흐린 배경 이미지 설정
Framer.Device.background.image = "photo.jpg"
Framer.Device.background.blur = 10
 
# 현재 기기의 화면 크기 가져오기
print Framer.Device.screen.width  # Output: 640 
print Framer.Device.screen.height # Output: 480

화면에 맞게 보기는 콘텐츠를 픽셀이 맞지 않게 렌더링하여 화면 비율을 유지할 수도 있습니다. 하지만 50%, 100% 등 기타 단위로 볼 때 콘텐츠가 항상 픽셀에 맞춰 보여집니다. 브라우저에서 지원하는 경우, 기기 이미지가 JPEG2000으로 압축되어 이미지 크기가 줄어듭니다.

기기 위치를 변경하려면 일반 레이어와 마찬가지로 화면 및 기기의 x 및 y 값을 변경하세요.


# 기기 위치 설정
Framer.Device.screen.x = 50
Framer.Device.x = 50
 
# 현재 기기 위치에서 변경
Framer.Device.screen.x += 50
Framer.Device.x += 50

Device.customize(deviceType, screenWidth, screenHeight, deviceImage, deviceImageWidth, deviceImageHeight, devicePixelRatio)

여러분이 원하는 이미지와 크기로 새로운 기기를 정의해보세요. 기기가 스마트폰 또는 태블릿 인 경우, 유형을 지정해서 미리보기하면 기기 이미지가 표시되지 않도록 할 수 있습니다.


# 커스터마이즈한 기기를 정의
Framer.Device.customize
    deviceType: Framer.Device.Type.Tablet
    screenWidth: 720
    screenHeight: 1024
    deviceImage: "http://f.cl.ly/items/001L0v3c1f120t0p2z24/custom.png"
    deviceImageWidth: 800
    deviceImageHeight: 1214

Device.fullScreen <boolean>

deviceType과 상관없이 전체 화면을 렌더링하도록 설정합니다. true 또는 false로 설정하세요.


# 기기를 전체화면으로 렌더링하기
Framer.Device.fullScreen = true

Device.deviceScale <number / string>

기기의 크기를 정의합니다. 지원되는 크기 변경 범위는 0.5 ~ 1배입니다.


Framer.Device.deviceScale = 0.5

Device.setDeviceScale(scale, animate)

기기의 크기와 변경 시 애니메이션 (true 또는 false)을 지정할 수 있습니다. 지원되는 크기 변경 범위는 0.5 ~ 1배입니다.


# 기기 크기와 애니메이션 설정
Framer.Device.setDeviceScale(0.5, true)

Device.contentScale

커스터마이즈한 기기까지 포함한 콘텐츠의 크기. 지원되는 크기 변경 범위는 0.5 ~ 1배입니다.


# 콘텐츠 크기 설정
Framer.Device.contentScale = 0.5

Device.setContentScale(scale, animate)

콘텐츠의 크기와 변경 시 애니메이션 (true 또는 false)을 지정할 수 있습니다. 지원되는 크기 변경 범위는 0.5 ~ 1배입니다.


# 콘텐츠 크기와 애니메이션 설정
Framer.Device.setContentScale(0.5, true)

Device.orientation

기기의 방향. 지원되는 방향 값은 0과 90 (세로 - 포트레이트, 가로 - 랜드스케이프)입니다.


Framer.Device.orientation = 90

Device.setOrientation(orientation, animate)

기기의 방향 및 변경 시 애니메이션 (true 또는 false)을 설정합니다. 지원되는 방향 값은 0과 90 (세로 및 가로 방향)입니다.


# 기기방향과 애니메이션 설정
Framer.Device.setOrientation(90, true)

Device.orientationName

기기 방향을 이름으로 설정하십시오. 유효한 옵션은 "portrait"와 "landscape"입니다. 세로와 가로 값은 기기 방향을 각각 0과 90으로 설정하는 것과 같습니다.


# 기기 방향을 세로, 아니면 가로로 설정
Framer.Device.orientationName = "landscape"
Framer.Device.orientationName = "portrait"

Device.rotateLeft()

기기를 왼쪽(반시계방향)으로 돌립니다.


Framer.Device.rotateLeft()

Device.rotateRight()

기기를 오른쪽(시계방향)으로 돌립니다.


Framer.Device.rotateRight()
