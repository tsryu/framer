레이어는 가로 및 / 또는 세로로 드래그 할 수 있습니다. 그들은 모멘텀을 가지고 던져질 수 있습니다. 드래그가 끝나는 위치를 제어하려면 드래그 할 수있는 특정 영역을 정의하면 됩니다. 옵션으로 영역 밖으로 드래그할 수 있지만, 다시 바운스되어 돌아옵니다.

layer.draggable.enabled <boolean>

레이어에 드래그를 사용합니다.


layerA = new Layer
layerA.draggable.enabled = true

layer.draggable.horizontal <boolean>

수평 드래그를 활성화 또는 비활성화합니다.


layerA = new Layer
layerA.draggable.enabled = true
 
# 수평 드래그를 비활성화
layerA.draggable.horizontal = false

layer.draggable.vertical <boolean>

수직 드래그를 활성화 또는 비활성화합니다.


layerA = new Layer
layerA.draggable.enabled = true
 
# 수직 드래그를 비활성화
layerA.draggable.vertical = false

layer.draggable.speedX <number>

수평 드래그 속도를 수정합니다. 값은 마우스 이동 픽셀 단위입니다. 기본값은 1입니다. 1보다 낮게 설정하면 드래그가 마우스 이동보다 느려지고, 반대로 높게 설정하면 빨라집니다. 값을 0으로 설정하여 수평 드래그를 비활성화할 수 있습니다.


layerA = new Layer
layerA.draggable.enabled = true
 
# 수평 드래그 느리게하기
layerA.draggable.speedX = 0.1
 
# 수평 드래그 빠르게하기
layerA.draggable.speedX = 10

layer.draggable.speedY <number>

수직 드래그 속도를 수정합니다. 값은 마우스 이동 픽셀 단위입니다. 기본값은 1입니다. 1보다 낮게 설정하면 드래그가 마우스 이동보다 느려지고, 반대로 높게 설정하면 빨라집니다. 값을 0으로 설정하여 수직 드래그를 비활성화할 수 있습니다.


layerA = new Layer
layerA.draggable.enabled = true
 
# 수직 드래그 느리게하기
layerA.draggable.speedY = 0.1
 
# 수직 드래그 빠르게하기
layerA.draggable.speedY = 10

layer.draggable.constraints <object>

레이어를 드래그 할 수 있는 영역을 제한합니다. x 및 y 속성도 제약 조건으로 설정하면 레이어가 DragStart시 해당 위치로 이동합니다. 레이어는 클릭 시에 새 포인트로 애니메이션됩니다.


layerA = new Layer
layerA.draggable.enabled = true
 
# 드래그 제한을 설정합니다.
layerA.draggable.constraints =
    x: 0
    y: 0
    width: 200
    height: 200

layer.draggable.constraintsOffset <object>

드래그 제한 영역 기준으로 오프셋 위치를 가져옵니다. 드래그 제한 영역을 {x : 100, y : 100}으로 설정해도, 시작시 x : 0, y : 0에 배치됩니다. 하지만 드래그한 후에는 드래그 제한영역 위치로 바운드됩니다. 이 오프셋은 constraintsOffset을 사용하여 측정 할 수 있습니다.


layerA = new Layer
layerA.draggable.enabled = true
 
# 드래그 제한 영역 설정
layerA.draggable.constraints =
    x: 100
    y: 100
    width: 200
    height: 200
 
# constraintsOffset 얻기
print layerA.draggable.constraintsOffset
 
# { x:-100, y:-100 }을 반환

layer.draggable.isBeyondConstraints <object>

드래그 가능한 레이어가 드래그 제한 영역 밖을 나갔는지 측정합니다. (읽기 전용)


layerA = new Layer
layerA.draggable.enabled = true
 
# 드래그 제한 영역 설정
layerA.draggable.constraints =
    x: 100
    y: 100
    width: 400
    height: 400
 
# 움직임을 감지해, 레이어가 드래그 제한 영역을 넘었는지 체크
layerA.on Events.Move, ->
    print layerA.draggable.isBeyondConstraints

layer.draggable.overdrag <boolean>

드래그 제한 영역 이상으로 드래그를 가능 혹은 불가능하게 설정합니다.


layerA = new Layer
layerA.draggable.enabled = true
 
layerA.draggable.constraints =
    x: 0
    y: 0
    width: 200
    height: 200
 
# 드래그 제한 영역을 넘어 드래그할 수 있도록 설정합니다.
layerA.draggable.overdrag = false

layer.draggable.overdragScale <number>

드래그 제한영역 밖으로 드래그했을 때 저항을 설정합니다. 0에서 1 사이의 값을 가지며, 기본값은 0.5입니다.


layerA = new Layer
layerA.draggable.enabled = true
 
layerA.draggable.constraints =
    x: 0
    y: 0
    width: 200
    height: 200

# 드래그 제한 영역 밖으로 드래그했을 때 저항을 늘린다
layerA.draggable.overdragScale = 0.25

layer.draggable.momentum <boolean>

모멘텀/관성을 활성화 또는 비활성화합니다. 기본적으로 활성화 되어있습니다.


layerA = new Layer
layerA.draggable.enabled = true
 
# 운동량을 비활성화
layerA.draggable.momentum = false

layer.draggable.momentumOptions <object>

DragEnd(드래그를 놨을 때)에 쓰이는 운동량 옵션


layerA = new Layer
layerA.draggable.enabled = true
 
# 모멘텀의 마찰과 저항을 정의
layerA.draggable.momentumOptions =
    friction: 2.1
    tolerance: 0.1

layer.draggable.velocity <object>

드래그 가능한 레이어의 현재 속도. 속도는 읽기 전용입니다.


layerA = new Layer
layerA.draggable.enabled = true
 
# 드래그 이동 시, x와 y의 속도를 보여준다  
layerA.draggable.on Events.DragMove, ->
    print layerA.draggable.velocity

layer.draggable.direction <string>

드래그 방향을 보여줍니다. ‘위', ‘아래', ‘왼쪽', ‘오른쪽'을 반환합니다. (읽기 전용)


layerA = new Layer
layerA.draggable.enabled = true
 
# 현재 방향을 출력합니다.
layerA.on Events.DragMove, ->
    print layerA.draggable.direction

layer.draggable.angle <number>

드래그 가능한 레이어의 각도를 보여줍니다.


layerA = new Layer
layerA.draggable.enabled = true
 
# 현재 각도 출력
layerA.on Events.DragMove, ->
    print layerA.draggable.angle

layer.draggable.updatePosition(point)

설정하기 전에 최종값을 재정의하는 기능. 드래그 가능 객체에 자신만의 기능을 추가할 수 있습니다. 이렇게 하면 특정 거리 사이에 이동 가능한 드래그 가능한 레이어를 만들 수 있습니다.

인수

  1. point — 객체형태, x와 y 속성

layerA = new Layer
layerA.draggable.enabled = true
 
# 설정된 숫자로 반올림
round = (number, nearest) ->
    Math.round(number / nearest) * nearest
 
# 20px 단위로 드래그
layerA.draggable.updatePosition = (point) ->
    point.x = round(point.x, 20)
    point.y = round(point.y, 20)
    return point

layer.draggable.directionLock <boolean>

특정 임곗값 이후 수평 / 수직 방향으로 이동합니다.


layerA = new Layer
layerA.draggable.enabled = true
 
# 한 번에 한 방향으로 만 드래그 허용
layerA.draggable.directionLock = true

layer.draggable.directionLockThreshold <object>

잠금 방향에 대한 임곗값입니다. x 및 y 값은 잠그기 전에 특정 방향으로 끌 수 있는 거리를 나타냅니다.


layerA = new Layer
layerA.draggable.enabled = true
 
# 가로로 50px 드래그한 후에 이동 가능
# 즉시 수직으로 이동 가능
layerA.draggable.directionLock = true
 
layerA.draggable.directionLockThreshold =
    x: 50
    y: 0

layer.draggable.pixelAlign <boolean>

드래그하는 동안 픽셀단위로 이동하여 서브 픽셀 안티 앨리어싱을 방지합니다.


layerA = new Layer
layerA.draggable.enabled = true
 
# 픽셀단위로 드래그
layerA.draggable.pixelAlign = true

layer.draggable.propagateEvents <boolean>

드래그 가능한 레이어의 propagateEvents 속성을 설정합니다. 기본적으로 true로 설정해보세요. 이 기능은 ScrollComponents 또는 PageComponent 또는 드래그 가능한 레이어로 작업 할 때 유용합니다.

scroll.content 레이어 내에 드래그 가능한 레이어를 만들고 싶다고 가정 해 보겠습니다. 기본적으로 레이어를 이동하면 scroll.content도 이동합니다. 이는 두 레이어가 드래그 이벤트를 인지하기 때문입니다.

드래그 가능한 자식 레이어가 부모 레이어에게 이벤트를 전달하지 못하게하려면 propagateEvents를 false로 설정해보세요. 이는 중첩 된 모든 드래그 가능한 레이어에 적용됩니다.


scroll = new ScrollComponent
    width: Screen.width,
    height: Screen.height
 
scroll.content.backgroundColor = "#28affa"
 
layerA = new Layer
    parent: scroll.content,
    backgroundColor: "#fff"
 
layerA.draggable.enabled = true
 
# propagateEvents를 false로 설정하면, ScrollComponent 내에서 스크롤하지 않고 layerA를 드래그할 수 있습니다.
 
layerA.draggable.propagateEvents = false
