이벤트는 감지하는 기능입니다. 터치 또는 클릭과 같은 사용자 인터랙션으로 애니메이션을 시작할 수 있습니다. 거의 모든 객체는 프레이머에서 이벤트를 지원하지만, 주로 레이어에 사용합니다.

이벤트가 호출될 때 첫 번째 인수는 이벤트 정보입니다. 이벤트에 따라 마우스 위치, 마우스 변위값 등을 포함 할 수 있습니다. 두 번째 인수는 항상 이벤트가 발생한 레이어입니다.

이벤트를 감지할 때, on 함수를 사용할 수 있습니다.


layerA = new Layer
layerA.name = "Layer A"
 
layerA.on Events.Click, (event, layer) ->
    print "Clicked", layer.name
 
# 결과: “Clicked”, “Layer A”

이벤트 감지를 중지하려면 off 함수를 사용할 수 있습니다.


layerA = new Layer
layerA.name = "Layer A"
 
clickHandler = (event, layer) ->
    print "Clicked", layer.name
 
layerA.on(Events.Click, clickHandler)
layerA.off(Events.Click, clickHandler)

Change Events

"change" 이벤트를 사용하면 변경되는 속성들을 감지할 수 있습니다. 다음은 "change" 이벤트로 감지할 수 있는 전체 속성의 개요입니다.

  1. "change:x" — 새로운 x 위치
  2. "change:y" — 새로운 y 위치
  3. "change:point" — 새로운 x 또는 y 위치
  4. "change:width" — 새로운 너빗값
  5. "change:height" — 새로운 높이값
  6. "change:size" — 새로운 너비와 높이값
  7. "change:frame" — 새로운 x값, y값, 너빗값, 높이값
  8. "change:scale" — 새로운 크기변경 값
  9. "change:rotation" — 새로운 회전 값
  10. "change:borderRadius" — 새로운 borderRadius 값
  11. "change:currentPage" — 새로운 currentPage 값
  12. "change:style" — 새로운 스타일 정의
  13. "change:html" — 새로운 html 정의
  14. "change:children" — 자식 레이어가 추가되거나 제거되었을 때
  15. "change:parent" — 새로운 부모 레이어가 추가되거나 제거되었을 때

예를 들어 애니메이션이 작동하는 동안 레이어의 x 위치를 가져올 수 있습니다. 정확한 서브픽셀 값만 반환한다는 점을 유의하세요.


layerA = new Layer
 
layerA.animate
    x: 100
 
layerA.on "change:x", ->
    print layerA.x

"change" 이벤트는 modulate(프레이머 예약어)로 두 레이어를 연결해서 속성 변경할 수 있습니다. 아래 예와 같이 첫 번째 레이어를 회전시키는데요. 반환 된 값은 두 번째 레이어를 가로로 이동하는 데 사용됩니다.


layerA = new Layer
layerB = new Layer
    x: 100
 
# 0에서 180도까지 레이어 A를 회전합니다.
layerA.animate
    rotation: 180
 
# layerA의 회전 값이 변경될 때
layerA.on "change:rotation", ->
 
    # 값을 사용하여 layerB를 100에서 300으로 이동합니다.
    x = Utils.modulate(layerA.rotation, [0, 180], [100, 300], true)
    layerB.x = x

Gesture Event Properties

모든 동작은 다음 속성들을 사용하여 이벤트 객체를 받습니다. touchCenter, touchDistance, touchOffset, scale 및 rotation은 다중 터치 이벤트를 사용할 때만 값을 반환합니다.

Positioning

  1. event.point — 현재 x와 y의 위치
  2. event.start — x와 y의 시작위치
  3. event.previous — 이전 x와 y의 위치

layerA = new Layer
layerA.pinchable.enabled = true
 
layerA.on Events.Pinch, (event) ->
    print event.point

Offset

  1. event.offset — 현재 x와 y의 오프셋 위치
  2. event.offsetTime — 시작한 이후로의 현재 시간
  3. event.offsetAngle — 시작한 이후로의 현재 각도
  4. event.offsetDirection — 시작한 이후로의 현재 방향

layerA = new Layer
 
layerA.on Events.Pan, (event) ->
    print event.offset

Deltas

  1. event.delta — 마지막 이벤트 이후로 오프셋 위치
  2. event.deltaTime — 마지막 이벤트 이후로 걸린 시간
  3. event.deltaAngle — 마지막 이벤트 이후로 바뀐 각도
  4. event.deltaDirection — 마지막 이벤트 이후의 방향 변경

layerA = new Layer
 
layerA.on Events.Swipe, (event) ->
    print event.delta

Velocity & Force

  1. event.velocity — x와 y값의 현재 속도
  2. event.force — 현재 탭의 강도

layerA = new Layer
 
layerA.on Events.Swipe, (event) ->
    print event.velocity

Input

  1. event.fingers — 스크린에 닿아있는 손가락의 수
  2. event.touchCenter — 두 손가락 사이의 중앙 위치
  3. event.touchDistance — 두 손가락 사이의 거리
  4. event.touchOffset — 두 손가락 사이의 오프셋 위치

layerA = new Layer
 
layerA.on Events.Rotate, (event) ->
    print event.fingers

Scale & Rotation

  1. event.scale — 두 손가락의 크기변경 값
  2. event.scaleDirection — 현재 크기변경의 방향 (위 아니면 아래)
  3. event.rotation — 두 손가락의 회전 값

layerA = new Layer
layerA.pinchable.enabled = true
 
layerA.on Events.Pinch, (event) ->
    print event.scaleDirection

Events.touchEvent(event)

모바일에서 입력된 이벤트에서 터치 이벤트를 추출합니다.


layerA = new Layer
 
layerA.on Events.Click, (event, layer) ->
    myTouchEvent = Events.touchEvent(event)

Events.wrap(event)

주어진 DOM 요소를 감싸서 이벤트를 추적하고 필요에 따라 없앨 수 있습니다. 이벤트를 임의의 DOM 요소에 바인드하려면 이 항목을 사용해야합니다.


Events.wrap(window).addEventListener "resize", (event) ->
    print "Page is resizing"
