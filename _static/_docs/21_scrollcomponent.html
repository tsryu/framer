스크롤 컴포넌트는 내용을 스크롤하는 데 사용됩니다. 모멘텀과 스프링 모션을 구현하고, 커스터마이즈를 허용하며, 다양한 이벤트를 가지고 있습니다.

스크롤 컴포넌트는 두 개의 레이어로 구성됩니다. 스크롤 컴포넌트 자체는 내용을 마스킹하는 레이어입니다. 그것은 드래그 가능하며 드래그 제한영역이 활성화되어있는 콘텐츠 레이어를 가지고 있습니다. 이는 콘텐츠 레이어의 하위 레이어의 총 크기를 기반으로 콘텐츠 레이어의 크기를 자동으로 관리합니다.


# 스크롤 컴포넌트 만들기
scroll = new ScrollComponent
    width: 100
    height: 100
 
# 레이어 포함하기
layerA = new Layer
    parent: scroll.content

ScrollComponent.wrap()을 사용하여 스크롤 컴포넌트 내에 기존 레이어를 감쌀 수도 있습니다. 스크롤 컴포넌트는 내용과 부모 레이어 사이에 삽입됩니다. 이 기능은 스케치 또는 포토샵에서 디자인을 가져와서 레이어를 스크롤 가능하게 만들 때 유용합니다. 이곳에서 감싸기에 대해 더 배울 수 있습니다.


layerA = new Layer
    width: 300
    height: 300
 
scroll = ScrollComponent.wrap(layerA)

scroll.content <layer>

내용을 추가 할 레이어입니다. 내용을 추가하려면 새 레이어를 만들고 해당 부모 레이어(parent)를 scroll.content 레이어로 설정합니다. 콘텐츠가 스크롤 컴포넌트에 크기가 맞지 않은 경우, 마스킹됩니다.


scroll = new ScrollComponent

    height: 100
 
layerA = new Layer

    width: 100
    height: 200

scroll = new ScrollComponent

    height: 100
 
layerA = new Layer

    height: 200
 
scroll.contentInset =
    top: 20

scroll.closestContentLayer(originX, originY)

정의된 원점 기준으로 스크롤 컴포넌트에 가장 가까운 레이어를 가져옵니다. 원점은 0과 1 사이의 숫자로 정의됩니다. 여기서 (0,0)은 왼쪽 위 모서리이고, (0.5, 0.5) 가운데는, (1,1)은 오른쪽 아래 모서리입니다.

기본값은 (0,0)입니다. 즉, 스크롤 컴포넌트의 왼쪽 위부터 콘텐츠 레이어의 왼쪽 상단까지의 거리를 계산합니다.


scroll = new ScrollComponent
 
# 콘텐츠 레이어를 생성
layerA = new Layer
    parent: scroll.content
    name: "layerA"
    x: 0
    y: 0
 
layerB = new Layer
    parent: scroll.content
    name: "layerB"
    x: 50
    y: 50
 
# 중심점이 가장 가까운 레이어 가져 오기
# ScrollComponent의 중심점으로 보내기
print scroll.closestContentLayer(0.5, 0.5)

scroll.closestContentLayerForScrollPoint(originX, originY)

scroll.scrollToPoint(point, animate, animationOptions)

특정 지점으로 스크롤 합니다. 선택적으로 애니메이션을 적용할 수도 있습니다.

인수

  1. point — x와 y 속성을 가진 객체
  2. animate — 불리언(true 또는 false), 기본으로 true로 설정되어 있습니다. (선택)
  3. animationOptions — 커브, 시간, 딜레이 그리고 반복 속성의 객체 (선택)

scroll = new ScrollComponent
 
# 콘텐츠를 x: 200, y: 100로 스크롤
scroll.scrollToPoint(
    x: 200, y: 100
    true
    curve: Bezier.ease
)
 
# 아주 느리게 스크롤 하기
scroll.scrollToPoint(
    x: 200, y: 100
    true
    curve: Bezier.ease, time: 10
)

scroll.scrollToLayer(layer, originX, originY, animate, animationOptions)

특정 레이어로 스크롤합니다. scroll.content 레이어의 자식 레이어만 스크롤 할 수 있습니다.

인수

  1. layer — 레이어 객체
  2. originX — 숫자 형태, 0과 1 사이 (선택)
  3. originY — 숫자 형태, 0과 1 사이 (선택)
  4. animate — 불리언(true 또는 false) 형태, 기본으로 true로 설정되어 있습니다. (선택)
  5. animationOptions — 커브, 시간, 딜레이 그리고 반복 속성의 객체 (선택)

# 스크롤 컴포넌트 만들기
scroll = new ScrollComponent
    width: 500
    height: 500
 
# 배경 정의
layerA = new Layer
    x: 500
    y: 1000
    image: "bg.png"
    parent: scroll.content
 
# 이 레이어로 스크롤하기
scrollerA = new Layer
    parent: scroll.content
 
scroll.scrollToLayer(scrollerA)

originX 및 originY 인수는 스크롤 될 레이어 내의 지점을 정의합니다. 기본값은 0,0 - 왼쪽 위 모서리입니다.


scroll.scrollToLayer(
    layerA
    0.5, 0
    true
    time: 2
)

scroll.scrollToClosestLayer(originX, originY)

지정된 원점을 사용하여 가장 가까운 콘텐츠 레이어로 스크롤 합니다. 기본값인 상단 왼쪽 구석은 (0,0), 중앙은 (0.5, 0.5), 오른쪽 하단 구석은 (1,1) 입니다.

인수

  1. originX — 숫자 형태, 0과 1 사이
  2. originY — 숫자 형태, 0과 1 사이

scroll = new ScrollComponent
 
layerA = new Layer
    parent: scroll.content

    y: 75
 
# layerA 중앙으로 스크롤하기

scroll.wrap(layer)

새로운 ScrollComponent 내의 레이어를 감쌉니다. 이는 가져온 레이어에서 스크롤 가능한 레이어를 만드는 데 유용합니다.


# 스케치에서 가져오기
sketch = Framer.Importer.load "imported/Scrollable"

# “content” 레이어그룹을 감싸기
scroll = ScrollComponent.wrap(sketch.content)

이제 scroll은 ScrollComponent의 변수 이름입니다. 또한 ScrollContent 레이어 내에 레이어를 자동으로 감싸서 모멘텀, 오버 드래그 또는 바운스와 같은 특정 속성을 조정할 수 있습니다.


# 스케치 파일을 가져오기 
sketch = Framer.Importer.load "imported/Scrollable"
 
# “content” 레이어그룹을 감싸기
scroll = ScrollComponent.wrap(sketch.content)
 
# 스크롤 속성 변경
scroll.scrollHorizontal = false
scroll.speedY = 0.5
 
# scroll.content 속성을 변경
scroll.content.draggable.momentum = true
scroll.content.draggable.overdrag = false
scroll.content.draggable.bounce = false

scroll.updateContent()

내용의 크기와 드래그 제한영역을 다시 계산하고 업데이트합니다. 또한 contentInset도 다시 계산합니다.

콘텐츠 레이어의 크기를 변경하려는 경우, 이를 호출하여 그에 따라 ScrollComponent의 크기를 업데이트할 수 있습니다.


scroll = new ScrollComponent
 
# 스크롤 컴포넌트의 콘텐츠를 업데이트
scroll.updateContent()

scroll.copy()

내용과 속성을 포함하여 ScrollComponent를 복사합니다.


scroll = new ScrollComponent
 
layerA = new Layer
    parent: scroll.content
 
# 스크롤 컴포넌트를 복사
scrollB = scroll.copy()

scroll.propagateEvents <boolean>

드래그 가능한 내용 계층의 propagateEvents 속성을 설정합니다. 기본으로 true로 설정되어 있습니다. 중첩된 ScrollComponents 또는 PageComponent로 작업 할 때 유용합니다.

scroll.content 레이어 내에 드래그 가능한 레이어를 만든다고 가정해보겠습니다. 기본으로 레이어를 이동하면 scroll.content도 이동합니다. 이는 두 레이어가 드래그 이벤트를 수신하기 때문입니다.

드래그 가능한 자식 레이어가 부모 레이어에게 이벤트를 전달하지 못하게하려면 propagateEvents를 false로 설정합니다. 이는 중첩 된 모든 드래그 가능한 레이어에 적용됩니다.


scroll = new ScrollComponent
    width: Screen.width,
    height: Screen.height
 
scroll.content.backgroundColor = "#28affa"
 
layerA = new Layer
    parent: scroll.content,
    backgroundColor: "#fff"
 
layerA.draggable.enabled = true
 
# propagateEvents를 false로 설정하면, 스크롤 컴포넌트 내에서 스크롤하지 않고 레이어를 드래그할 수 있습니다.
layerA.draggable.propagateEvents = false
