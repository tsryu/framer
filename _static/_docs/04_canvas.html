캔버스는 현재 작업물 전체 크기를 픽셀 단위로 담고 있습니다. 창 크기를 조정하면 캔버스의 크기도 맞춰 변경됩니다.

Canvas.backgroundColor <string>

캔버스의 배경색을 설정합니다.


# 캔버스의 배경색을 바꿉니다.
	Canvas.backgroundColor = "#28affa"

Canvas.image <string>

캔버스의 배경 이미지를 설정합니다. 로컬 경로 또는 링크로 설정할 수 있습니다. 이미지 크기는 항상 캔버스에 맞춰 변경되지만, 비율이 깨지진 않습니다.


# Local images 로컬 이미지 불러오기
Canvas.image = "images/background.png"
 
# Hosted images URL로 불러오기
Canvas.image = "http://framerjs.com/background.png"

Canvas.width <number>

현재 작업물의 너비(픽셀)입니다. (읽기 전용)


print Canvas.width
# 결과: 640

Canvas.height <number>

현재 작업물의 높이(픽셀)입니다. (읽기 전용)


print Canvas.height
# 결과: 480

Canvas.size <object>

현재 작업물의 너비와 높이(픽셀 단위)입니다. (읽기 전용)


print Canvas.size
# 결과: { width:640, height: 480 }

Canvas.frame <object>

현재 작업물의 x, y, 너비 및 높이 (픽셀 단위)입니다. (읽기 전용)


print Canvas.frame
# 결과: { x:0, y:0, width:640, height: 480 }

Canvas.convertPointToScreen(point)

점을 캔버스에서 스크린 기준으로 변환합니다.


point =
    x: 20
    y: 40
pointInScreen = Canvas.convertPointToScreen(point)

Canvas.convertPointToLayer(point, layer)

점을 캔버스에서 레이어 기준으로 변환합니다.


point =
    x: 20
    y: 40
 
layer = new Layer
 
pointInLayer = Canvas.convertPointToLayer(point, layer)
