RangeSliderComponent는 두 개의 손잡이(knob)가 있는 범위 슬라이더를 만듭니다. 가격 필터와 같이 숫자 범위를 정의하고 편집하는 데 사용할 수 있습니다. slider, fill(현재 값을 나타내는 막대 혹은 다른 요소), minKnob, maxKnob 총 네 가지 레이어로 구성됩니다.


# 범위 슬라이더 생성
range = new RangeSliderComponent
    x: Align.center
    y: Align.center

슬라이더는 기본으로 어두운 fill 색상, 밝은 회색의 backgroundColor(배경색), 그리고 borderRadius(둥근 모서리)를 가지고 있습니다. fill의 모서리 둥근 정도는 슬라이더와 일치하기 때문에 한 번만 설정하면 됩니다.


# 디자인 커스터마이즈하기
range.backgroundColor = "#ddd"
range.borderRadius = 4

rangeSlider.min <number>

슬라이더의 최소 범위입니다. 기본으로 0으로 설정되어 있습니다.


# 범위 슬라이더 생성
range = new RangeSliderComponent
    x: Align.center
    y: Align.center
    min: 0
    max: 10

rangeSlider.max <number>

슬라이더의 최대 범위입니다. 기본으로 1로 설정되어 있습니다.


# 범위 슬라이더 생성
range = new RangeSliderComponent
    x: Align.center
    y: Align.center
    min: 0
    max: 10

rangeSlider.minValue <number>

슬라이더의 최솟값입니다. 기본으로 0으로 설정되어 있습니다.


# 범위 슬라이더 생성
range = new RangeSliderComponent
    x: Align.center
    y: Align.center
    minValue: 0

여러분은 최솟값이 변경되는 것만 체크해서 이벤트를 실행할 수 있습니다.


range.onMinValueChange ->
    print range.minValue

또한 값이 변경되는 것도 체크해서 이벤트를 실행할 수 있습니다.


range.onValueChange ->
    print range.minValue, range.maxValue

rangeSlider.maxValue <number>

슬라이더의 최댓값입니다. 기본으로 0.5으로 설정되어 있습니다.


# 범위 슬라이더 생성
range = new RangeSliderComponent
    x: Align.center
    y: Align.center
    maxValue: 0.5

여러분은 최댓값이 변경되는 것만 체크해서 이벤트를 실행할 수 있습니다.


range.onMaxValueChange ->
    print range.maxValue

또한 Value Change 이벤트로 값이 변경되는 것을 체크해서 이벤트를 실행할 수 있습니다.


range.onValueChange ->
    print range.minValue, range.maxValue

rangeSlider.knobSize <number>

양쪽 손잡이의 크기를 설정합니다. 이것은 자동으로 minKnob과 maxKnob의 너비와 높이로 설정됩니다. 기본값은 30입니다.


# 범위 슬라이더 생성
range = new RangeSliderComponent
    x: Align.center
    y: Align.center
 
# 손잡이의 크기를 바꿉니다
range.knobSize = 40

rangeSlider.fill <layer>

슬라이더의 fill 부분입니다. 너비와 높이는 슬라이더의 크기에 따라 자동으로 설정됩니다. 기본으로 backgroundColor는 #333으로 설정됩니다.


# 범위 슬라이더 생성
range = new RangeSliderComponent
    x: Align.center
    y: Align.center
 
# 색상을 파란색으로 설정합니다
range.fill.backgroundColor = "#00AAFF"

rangeSlider.minKnob <layer>

슬라이더의 최솟값에 영향을 주는 손잡이입니다.


# 범위 슬라이더 생성
range = new RangeSliderComponent
    x: Align.center
    y: Align.center

minKnob은 momentum이 활성화된 드래그 가능한 레이어입니다.


# 모멘텀 비활성화
range.minKnob.draggable.momentum = false

rangeSlider.maxKnob <layer>

슬라이더의 최댓값에 영향을 주는 손잡이입니다.


# 범위 슬라이더 생성
range = new RangeSliderComponent
    x: Align.center
    y: Align.center

maxKnob은 momentum이 활성화된 드래그 가능한 레이어입니다.


# 모멘텀 비활성화
range.maxKnob.draggable.momentum = false

rangeSlider.animateToMinValue(minValue, animationOptions)

특정 최솟값까지 자동으로 애니메이션을 적용합니다.

인수

  1. minValue — 숫자 형태, 최솟값을 설정합니다.
  2. animationOptions — 객체 형태, 커브, 시간, 딜레이 그리고 반복 속성을 설정합니다. (선택)

# 범위 슬라이더 생성
range = new RangeSliderComponent
    x: Align.center
    y: Align.center
 
# 1까지 애니메이션 적용
range.animateToMinValue(0.3)
 
# 커스터마이즈한 커브 애니메이션 적용
range.animateToMinValue(0.3, { curve: Spring })

rangeSlider.animateToMaxValue(maxValue, animationOptions)

특정 최댓값까지 자동으로 애니메이션을 적용합니다.

인수

  1. minValue — 숫자 형태, 최솟값을 설정합니다.
  2. animationOptions — 객체 형태, 커브, 시간, 딜레이 그리고 반복 속성을 설정합니다. (선택)

# 범위 슬라이더 생성
range = new RangeSliderComponent
    x: Align.center
    y: Align.center
 
# 1까지 애니메이션 적용
range.animateToMaxValue(1)
 
# 사용자 정의 커브 애니메이션 적용
range.animateToMaxValue(1, { curve: Spring })
