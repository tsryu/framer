TextLayer를 사용하면 프레이머에서 텍스트를 가공할 수 있습니다. 문자열 내에 텍스트를 정의하고 고유한 속성을 사용하여 스타일을 지정합니다. 거의 모든 속성 이름은 CSS 텍스트 스타일과 일치합니다. 서체를 변경하고, 스타일을 변경하고, 줄 높이를 조정하고, 내부여백, 그림자 등을 추가할 수 있습니다.

text.text <string>

텍스트 내용. 기본으로 color는 #888로 설정됩니다.


# TextLayer 만들기
title = new TextLayer
    text: "Title"

text.fontSize <number>

텍스트의 크기. 기본으로 40으로 설정됩니다.


# fontSize를 64px로 설정 
title = new TextLayer
    fontSize: 64

text.fontFamily <string>

텍스트의 서체. 기본적으로 프리뷰하고 있는 기기의 시스템 글꼴을 사용하도록 설정되어 있습니다.

  • 애플 기기: SF UI.
  • 구글 기기: Roboto.
  • 마이크로소프트 기기: Segoe UI.

# TextLayer 만들기
title = new TextLayer
    fontFamily: "-apple-system"

text.fontWeight <number>

텍스트의 굵기를 정의합니다. 기본으로 400으로 설정되어 있습니다.


# TextLayer 만들기
title = new TextLayer
    fontWeight: 400

text.fontStyle <string>

텍스트의 스타일을 정의합니다.


# 폰트를 이탤릭체로 설정
title = new TextLayer
    fontStyle: "italic"

여러분은 italic, bold 그리고 oblique 중에 선택할 수 있습니다. 


# 폰트 스타일을 볼드로 설정
title = new TextLayer
    fontStyle: "bold"

text.font <string>

한 줄 안에 모든 폰트 스타일을 정의합니다. 이 속성은 다음 순서대로 설정할 수 있습니다 : "fontStyle fontWeight fontSize / lineHeight fontFamily".


# 폰트 스타일을 한 줄에 모두 정의하기
title = new TextLayer
    font: "300 64px/1 -apple-system, Helvetica Neue"

fontSize 및 fontFamily 속성은 필수 항목이라는 점을 유의하세요.


# 폰트 사이즈와 패밀리를 설정
title = new TextLayer
    font: "64px -apple-system"

text.padding <object>

텍스트 레이어의 내부여백(padding)입니다. 기본적으로 0으로 설정됩니다.


# 내부여백 설정하기
title = new TextLayer
    padding: 40

또한 내부여백을 방향별로 정의할 수 있습니다.


# 내부여백을 방향별로 정의
title = new TextLayer
    padding:
        top: 40
        left: 20
        bottom: 40
        right: 20

왼쪽 및 오른쪽 내부여백은 horizontal를 사용하여 정의할 수 있습니다. 또한 상단 및 하단 내부여백은 vertical를 사용하여 정의할 수도 있습니다.


# 가로 및 세로 내부여백 설정
title = new TextLayer
    padding:
        horizontal: 40
        vertical: 80

text.lineHeight <number>

텍스트의 줄 높이입니다. 기본으로 1.25로 설정되어 있습니다. 즉, fontSize를 40px로 설정하면 줄 높이가 50px가됩니다.


# 줄 높이를 설정
title = new TextLayer
    lineHeight: 1.5

text.letterSpacing <number>

글자의 문자의 간격 (픽셀 단위)입니다. 기본으로 0으로 설정됩니다.


# 문자 간격 설정
title = new TextLayer
    letterSpacing: 2

text.wordSpacing <number>

단어의 문자 간격 (픽셀 단위)입니다. 기본으로 0으로 설정됩니다.


# 단어의 문자 간격 설정
title = new TextLayer
    wordSpacing: 10

text.textAlign <string>

텍스트의 정렬을 정의합니다.


# 텍스트 중앙정렬
title = new TextLayer
    textAlign: "center"

left, right 그리고 center 중에 선택할 수 있습니다.


# 텍스트를 정렬하는 여러 가지 방법
title.textAlign = "left"
title.textAlign = "right"
title.textAlign = "center"

대안으로, Align 클래스를 사용할 수도 있습니다.


# 텍스트 중앙정렬
title = new TextLayer
    textAlign: Align.center

text.textTransform <string>

텍스트의 대문자 혹은 소문자 사용을 설정합니다.


# 모든 문자를 대문자로 바꿉니다.
title = new TextLayer
    textTransform: "uppercase"

uppercase, lowercase 그리고 capitalize 중에 선택할 수 있습니다.


# 대문자, 혹은 소문자 사용을 설정하는 여러 가지 방법
# 전체 대문자
title.textTransform = "uppercase”
# 전체 소문자
title.textTransform = "lowercase"
# 첫 번째 글자만 대문자
title.textTransform = "capitalize"

text.textDecoration <string>

글자에 밑줄(underline), 취소선(line-through)과 같은 꾸밈을 추가합니다.


# 텍스트에 밑줄 추가
title = new TextLayer
    textDecoration: "underline"

underline, overline 그리고 line-through 중에 선택할 수 있습니다.


# 텍스트에 윗줄 추가
title = new TextLayer
    textDecoration: "overline"

text.textIndent <number>

픽셀 단위로 텍스트의 첫 번째 단락을 들여쓰기합니다.


# 텍스트를 20 픽셀 들여쓰기
title = new TextLayer
    textIndent: 20

text.textOverflow <string>

넘치는 텍스트 내용에 줄임표를 추가합니다.


# 한 줄 줄임표 추가하기
paragraph = new TextLayer
    textOverflow: "ellipsis"
    text:
        """
        Lorem ipsum dolor sit amet, consectetur adipiscing elit,
        sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
        """

height(높이)를 설정하면 2줄 이상의 글도 줄임표를 추가할 수 있습니다.


# 2줄 이상의 글 줄임표 추가
paragraph = new TextLayer
    textOverflow: "ellipsis"
    height: 100
    text:
        """
        Lorem ipsum dolor sit amet, consectetur adipiscing elit,
        sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
        """

text.direction <string>

텍스트의 방향입니다. 텍스트의 방향입니다. 기본으로 left-to-right(왼쪽에서 오른쪽으로)로 설정되어 있습니다.


# 텍스트 방향 정하기
title = new TextLayer
    direction: "right-to-left"

right-to-left (rtl) 과 left-to-right (ltr) 중에 선택할 수 있습니다.


# 텍스트 방향 정하기
title = new TextLayer
    direction: "rtl"

text.truncate

textOverflow(넘치는 텍스트)에 ellipsis(줄임표)를 더하는 단축코드입니다.


# 한 줄 줄임표 더하기
paragraph = new TextLayer
    truncate: true
    text:
        """
        Lorem ipsum dolor sit amet, consectetur adipiscing elit,
        sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
        """

height(높이)를 설정하면 2줄 이상의 글도 줄임표를 추가할 수 있습니다.


# 2줄 이상의 글 줄임표 추가
paragraph = new TextLayer
    truncate: true
    height: 100
    text:
        """
        Lorem ipsum dolor sit amet, consectetur adipiscing elit,
        sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
        """

text.shadowX <number>

x 축의 그림자 방향을 정의합니다. 양수 값은 레이어의 오른쪽 가장자리에서 그림자를 생성하지만 음수 값이면 왼쪽 가장자리에서 그림자를 생성합니다.


# 가로 텍스트 그림자 설정
title = new TextLayer
    shadowX: 10

text.shadowY <number>

y 축의 그림자 방향을 정의합니다. 양수 값은 레이어의 아래쪽 가장자리에서 그림자를 생성하지만 음수 값을 지정하면 위쪽 가장자리에서 그림자가 생성됩니다.


# 세로 텍스트 그림자 설정
title = new TextLayer
    shadowY: 10

text.shadowBlur <number>

shadowX 또는 shadowY 속성에 가우시안 블러를 추가합니다. shadowBlur는 숫자로 정의됩니다. 기본값은 0입니다.


# 텍스트 그림자를 설정합니다.
title = new TextLayer
    shadowY: 1
    shadowBlur: 4

text.shadowColor <string>

레이어 그림자의 색상을 설정합니다. 색상은 CSS 색상 형식의 문자열로 표현됩니다.


# 텍스트 그림자 및 색상 설정
title = new TextLayer
    shadowY: 10
    shadowColor: "rgba(0,0,0,0.2)"
