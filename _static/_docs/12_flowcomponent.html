FlowComponent를 사용하면 여러 화면 사이를 전환하고 이동할 수 있습니다. 새로운 레이어로 전환하는 showNext와 이전 레이어를 거쳐 순환하는 showPrevious의 두 가지 기본 기능을 기반으로 합니다. 또한 모달과 같은 오버레이나 탭 바와 같은 고정 요소를 디자인하는 데 사용할 수 있습니다. FlowComponent는 Transition 이벤트를 발생시킵니다. 트랜지션 이벤트는 이곳에서 더 자세히 알아보세요.

속성

  1. layer — 레이어 객체, 대상 레이어
  2. options — 객체 형태, 커브와 시간과 같은 모든 애니메이션 옵션을 가진 객체입니다. (선택)

# 레이어 만들기
layerA = new Layer
    size: Screen.size
 
# 플로우 컴포넌트 만들기
flow = new FlowComponent
 
# 레이어 보여주기
flow.showNext(layerA, animate: true)

flow.showNext(layer, options)

새 레이어로 전환합니다. 기본으로 FlowComponent의 첫 번째 레이어를 제외한 모든 레이어에 애니메이션을 적용합니다. 레이어의 너비 또는 높이가 부모 레이어의 너비 또는 높이를 초과하면 스크롤 할 수 있게 됩니다. 또한 수직 또는 수평 스크롤 가능 여부를 자동으로 감지합니다.

인수

  1. layer — 레이어 객체
  2. options.animate — 불리언 형태(true 혹은 false로 입력), 레이어가 애니메이션으로 표시 여부를 설정합니다. (선택)
  3. options.scroll — 불리언 형태(true 혹은 false로 입력), 레이어가 스크롤 가능 여부를 설정합니다. (선택)

# 레이어 만들기
layerA = new Layer
    size: Screen.size
 
# 플로우 컴포넌트 만들기
flow = new FlowComponent
 
# 레이어 보여주기
flow.showNext(layerA)

애니메이트를 false로 설정하면 연결 애니메이션 없이 레이어 사이를 전 할 수 있습니다.


# 레이어 만들기
layerA = new Layer
    size: Screen.size
    backgroundColor: "#00AAFF"
 
layerB = new Layer
    size: Screen.size
    backgroundColor: "#FFCC33"
 
# 플로우 컴포넌트를 만들고 레이어를 보여주기
flow = new FlowComponent
flow.showNext(layerA)
 
# 클릭시 즉시 layerB 표시
layerA.onClick ->
    flow.showNext(layerB, animate: false)

flow.showPrevious(options)

이전 레이어로 전환합니다. 기본적으로 레이어에 애니메이션이 적용됩니다.

인수

  1. options.animate — 불리언 형태(true 혹은 false로 입력), 레이어가 애니메이션 적용 여부를 설정합니다. (선택)
  2. options.scroll — 불리언 형태(true 혹은 false로 입력), 레이어가 스크롤 가능 여부를 설정합니다. (선택)

# 레이어 만들기
layerA = new Layer
layerB = new Layer
 
# 플로우 컴포넌트 만들기
flow = new FlowComponent
flow.showNext(layerA)
 
# 클릭시 즉시 layerA로 바꾸기
layerA.onClick ->
    flow.showNext(layerB)
 
# 클릭시 이전으로 돌아오기
layerB.onClick ->
    flow.showPrevious()

  3. options.scroll — 불리언 형태(true 혹은 false로 입력), 레이어가 스크롤 가능 여부를 설정합니다. (선택)
  4. options.modal — 불리언 형태(true 혹은 false로 입력), 오버레이가 클릭 가능 여부를 설정합니다. (선택)

modal 인수는 기본적으로 false로 설정됩니다. 오버레이를 클릭 할 수 있고 이전 화면으로 되돌리려면 불리언을 true로 활성화 할 수 있습니다.


# 레이어 만들기

    backgroundColor: "#00AAFF"
    size: Screen.size
 
layerB = new Layer
    borderRadius: 40
    backgroundColor: "#FFF"
    size: 500
 
# 플로우 컴포넌트 만들기
flow = new FlowComponent
flow.showNext(layerA)
 
# 모달 보여주기, 오버레이는 클릭 불가하게 설정
layerA.onClick ->

flow.transition(layer, transition, options)

트랜지션을 커스터마이즈해서 만듭니다. 트랜지션은 상태(state)를 사용하여 앞뒤로 순환합니다. 두 가지 상태 (뒤와 앞)가 있는 세 개의 레이어 (현재, 다음 및 배경)가 있습니다. 특정 상태를 정의하지 않으면 FlowComponent는 해당 레이어에 애니메이션을 적용을 원하지 않는 것으로 가정합니다.

인수

  1. layer — 레이어 객체
  2. options.animate — 불리언 형태(true 혹은 false로 입력), 레이어가 애니메이션 적용 여부를 설정합니다. (선택)
  3. options.scroll — 불리언 형태(true 혹은 false로 입력), 레이어가 스크롤 가능 여부를 설정합니다. (선택)
  4. options.modal — 불리언 형태(true 혹은 false로 입력), 오버레이가 클릭 가능 여부를 설정합니다. (선택)

# 커스터마이즈한 트랜지션
scaleTransition = (nav, layerA, layerB, overlay) ->
    transition =
        layerA:
            show:
                scale: 1.0
                opacity: 1
            hide:
                scale: 0.5
                opacity: 0
        layerB:
            show:
                scale: 1.0
                opacity: 1
            hide:
                scale: 0.5
                opacity: 0
 
# 레이어 만들기
layerA = new Layer
    backgroundColor: "#00AAFF"
    size: Screen.size
 
layerB = new Layer
    backgroundColor: "#FFCC33"
    size: Screen.size
 
# 플로우 컴포넌트 만들기
flow = new FlowComponent
flow.showNext(layerA)
 
# layerB로 커스터마이즈한 트랜지션 적용
layerA.onClick ->
    flow.transition(layerB, scaleTransition)

커스텀 트랜지션은 상태(states)가 있는 객체를 반환하는 함수입니다. 함수 내에서 현재 FlowComponent, layerA, layerB 및 overlay 인수에 액세스 할 수 있습니다. layerA, layerB 또는 overlay에 대한 상태를 전달하지 않으면 전환시 그대로 있는 상태로 유지됩니다.

flow.header <layer>

고정된 네비게이션 바처럼 스크롤 가능한 화면에 고정 헤더(sticky header)를 추가합니다.


# 네비게이션 레이어 만들기
navBar = new Layer
 
# 플로우 컴포넌트 만들기
flow = new FlowComponent
 
# 레이어를 상단에 고정
flow.header = navBar

flow.footer <layer>

고정된 탭 바처럼 스크롤 가능한 화면에 고정 푸터(sticky footer)를 추가합니다.


# 탭 바 레이어를 만들기
tabBar = new Layer
 
# 플로우 컴포넌트를 만들기
flow = new FlowComponent
 
# 레이어를 하단에 고정
flow.footer = tabBar
