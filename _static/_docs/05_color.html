Color 객체는 색상을 정의, 감지, 수정 및 혼합하는 데 사용할 수 있습니다. 색상은 문자열 값 또는 객체를 사용하여 정의할 수 있습니다. 모든 색상은 r, g, b, h, s, l 및 a 값을 사용하여 Color 객체로 변환됩니다.


bg = new BackgroundLayer
        backgroundColor: "#28affa"
     
    print bg.backgroundColor
    # Color "#28affa"

지원하는 색상 단위: CSS 색상 이름, HEX, RGB, RGBA, HSL and HSLA.


# Multiple ways to define the same color: 여러 가지 방법으로 하나의 색상을 정의하는 예시
blue = new Color("blue")
blue = new Color("#28AFFA")
blue = new Color("rgb(255, 0, 102)")
blue = new Color("rgba(255, 0, 102, 1)")
blue = new Color("hsl(201, 95, 57)")
blue = new Color("hsla(201, 95, 57, 1)")

새로운 Color 객체를 만들고 문자열이나 객체를 전달할 수도 있습니다.


# 16진수 문자열로 색상을 정의
bg = new BackgroundLayer
    backgroundColor: new Color("#fff")
 
# RGB 객체로 색상 정의
layerA = new Layer
    backgroundColor: new Color(r: 255, g: 255, b: 255)
 
# HSL 객체로 색상 정의
layerB = new Layer
    backgroundColor: new Color(h: 360, s: 1, l: 1, a: 1)

Color Models

레이어의 배경색, 텍스트 색상 및 그림자 색상을 애니메이션으로 만들 수 있습니다. 기본으로 색 변환에는 HUSL이 사용됩니다. colorModel 속성을 사용하여 애니메이션을 적용할 모델을 지정할 수 있습니다. 프레이머는 rgb, hsl 및 husl을 지원합니다.


bg = new BackgroundLayer
    backgroundColor: "blue"
 
# RGB로 애니메이션 주기 
bg.animate
    backgroundColor: "red"
    options:
        colorModel: "rgb"

color.lighten(amount)

흰색을 추가하고 밝게 된 색상을 반환합니다.

인수

  1. amount — 숫자 형태, 0부터 100까지 쓸 수 있습니다. 기본으로 10으로 설정됩니다.

# 새 색상 만들고 밝게 하기
blue = new Color("#28affa").lighten(20)
 
layerA = new Layer
    backgroundColor: blue

color.darken(amount)

검은 색을 추가하고 어두운색을 반환합니다.

인수

  1. amount — 숫자 형태, 0부터 100까지 쓸 수 있습니다. 기본으로 10으로 설정됩니다.

# 새 색상을 만들고 어둡게 만들기
blue = new Color("#28affa").darken(20)a
 
layerA = new Layer
    backgroundColor: blue

color.saturate(amount)

색상의 채도를 높일 수 있습니다.

인수

  1. amount — 숫자 형태, 0부터 100까지 쓸 수 있습니다. 기본으로 10으로 설정됩니다.

# 새 색상을 만들고, 채도 올리기
blue = new Color("#877DD7").saturate(100)    

layerA = new Layer
    backgroundColor: blue

color.desaturate(amount)

색상의 채도를 낮출 수 있습니다.

인수

  1. amount — 숫자 형태, 0부터 100까지 쓸 수 있습니다. 기본으로 10으로 설정됩니다.

# 새 색상을 만들고, 채도를 낮추기
blue = new Color("#28affa").desaturate(25)
 
layerA = new Layer
    backgroundColor: blue

color.grayscale()

완전히 채도가 떨어지는 색을 반환합니다.


# 새로운 색상 만들기
yellow = new Color("yellow")
 
# 무채색으로 변환하기
gray = yellow.grayscale()
 
layerA = new Layer
    backgroundColor: gray

color.gray(amount, alpha)

투명한 흰색 배경을 생성합니다.

인수

  1. amount — 흰색의 양을 나타내는 숫자
  2. alpha — 투명도를 나타내는 숫자 (선택)

layer = new Layer
    layer.backgroundColor = Color.gray(0.5)

color.alpha(amount)

색상의 알파(오퍼시티) 값을 늘립니다.

인수

  1. amount — 숫자 형태, 0부터 1까지 쓸 수 있습니다. (소수점 사용 가능) 기본으로 1로 설정됩니다.

# 새 색상을 만들고 오퍼시티 주기
blue = new Color("#28affa").alpha(.5)
 
layerA = new Layer
    backgroundColor: blue

Color.mix(colorA, colorB, fraction, limit, model)

사용자 가 입력한 두 가지 색상을 혼합합니다. fraction은 두 색상 사이의 분포를 정의하며 기본적으로 0.5로 설정됩니다.

인수

  1. colorA — 색상 형태, 첫 번째 색상
  2. colorB — 색상 형태, 두 번째 색상
  3. fraction — 색상 혼합 비율 (선택)
  4. limit — 불리언 형태(true 혹은 false로 입력), 기본 false로 되어있습니다. 색상의 범위 제한 여부 (선택)
  5. model — 문자열 형태, 색상을 혼합하는 색상 모델 선택

# 빨간색과 노란색을 혼합
orange = Color.mix("red", "yellow", 0.5)

limit는 색상이 범위를 벗어날 수 있는지 정의합니다. 이것은 Utils.modulate를 사용하여 색상 사이를 전환 할 때 적용 할 수 있습니다. 아래에서 limit는 true로 설정되어 전환을 두 번 이상 할 수 없습니다.


# 레이어 만들기
layerA = new Layer
    backgroundColor: "red"
 
# 드래그 가능하게 설정
layerA.draggable.enabled = true
 
# 레이어 이동 시, 레이어를 노란색으로 변경
layerA.on Events.Move, (offset) ->
 
    # 드래그 거리를 0에서 1사이로 측정
    fraction = Utils.modulate(offset.x, [0, Screen.width], [0,1], true)
 
    # 색상 혼합, limit를 true로 서정, HUSL로 색상 혼합
    layerA.backgroundColor =
        Color.mix("red", "yellow", fraction, true, "husl")

Color.isColor(value)

값이 유효한 색상 객체 또는 색상 문자열인지 확인합니다. true 또는 false를 반환합니다.

인수

  1. value — 객체 또는 문자열 형태, 색상을 나타냅니다

print Color.isColor(new Color "red") # true 반환
print Color.isColor("red") # true 반환

Color.isColorObject(value)

값이 유효한 색상 객체인지 확인합니다. true 또는 false를 반환합니다.

인수

  1. value — 객체 형태, 색상을 나타냅니다

print Color.isColorObject(new Color "red") # true 반환
print Color.isColorObject("red") # false 반환

Color.isColorString(value)

값이 색상 문자열인지 확인합니다. true 또는 false를 반환합니다.

인수

  1. value — 문자열 형태, 색상을 나타냅니다

print Color.isColorString("red") # true 반환
print Color.isColorString("#28affa") # true 반환

Color.toHexString()

색상을 16진수 문자열로 반환합니다.

인수

  1. value — 문자열 형태, 색상을 나타냅니다

blue = new Color("blue")
print blue.toHexString() # "#0000ff"

Color.toRgbString()

색상을 RGB 문자열로 반환합니다.

인수

  1. value — 객체 또는 문자열 형태, 색상을 나타냅니다

blue = new Color("blue")
print blue.toRgbString() # "rgb(0, 0, 255)"

Color.toHslString()

색상을 HSL 문자열로 반환합니다.

인수

  1. value — 색을 나타내는 객체 혹은 문자열

blue = new Color("blue")
print blue.toHslString() # "hsl(240, 100%, 50%)"
