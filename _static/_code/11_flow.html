FlowComponent를 사용하면 여러 화면을 신속하게 전환할 수 있습니다. 특히 정적인 화면을 완전히 인터랙티브한 화면으로 변환하는 데 유용합니다.

FlowComponent의 독특한 점은 여러분이 보고 있는 화면을 알고 있다는 것입니다. 기록을 추적하여 언제든지 앞뒤로 이동할 수 있습니다. 기본적으로 모바일 앱 같은 상세한 전환 효과가 있습니다. 화면을 전환할 때 현재 화면이 자동으로 화면 밖으로 움직입니다. 모달팝업과 같은 오버레이를 추가할 때 배경에는 반투명의 어두운 오버레이가 적용됩니다.

FlowComponent에 레이어를 추가하는 것으로 시작해봅시다. 이는 PageComponent에 레이어를 추가하는 것과 매우 유사하며, showNext 함수를 사용하여 쉽게 수행할 수 있습니다. 추가한 첫 번째 레이어는 애니메이션이 추가되지 않습니다.


# 레이어 만들기
layerA = new Layer
 
# FlowComponent 만들기, layerA 보여주기 
flow = new FlowComponent
flow.showNext(layerA)

화면 전환하기

showNext 함수를 사용하여 다음 화면으로 이동할 수 있습니다.



# 레이어 만들기
layerA = new Layer
layerB = new Layer
 
# FlowComponent 만들기, 레이어 보여주기 
flow = new FlowComponent
flow.showNext(layerA)
 
# 클릭으로 화면 전환하기
layerA.onClick ->
    flow.showNext(layerB)

프레이머는 현재보고있는 화면을 추적합니다. showPrevious 함수를 사용하여 이전 화면으로 다시 전환할 수 있습니다.



# 다음 화면으로 전환하기
layerA.onClick ->
    flow.showNext(layerB)
 
# 이전 화면으로 전환하기
layerB.onClick ->
    flow.showPrevious()

오버레이

showOverlay 함수 중 하나로 어떤 레이어든 오버레이 할 수 있습니다. 오버레이하게 되면 현재 화면은 반투명한 어두운 배경이 적용됩니다. 이것은 사이드 바 또는 액션 시트(action sheet)와 같이 더 작은 레이어를 오버레이할 때 더욱 잘 보입니다. 다음 함수로 오버레이의 위치를 지정할 수 있습니다.

  • showOverlayCenter
  • showOverlayTop
  • showOverlayRight
  • showOverlayBottom
  • showOverlayLeft


# FlowComponent 만들기 
flow = new FlowComponent
 
# 모달과 버튼은 디자인 탭에서 설정
 
# 오버레이 보여주기
button.onClick ->
    flow.showOverlayCenter(modal)

스크롤

FlowComponent의 높이 또는 너비를 초과하는 자식 레이어는 자동으로 스크롤 할 수 있게 됩니다. 높이가 더 높으면 세로로 스크롤 할 수 있고 너비가 더 넓으면 가로로 스크롤 할 수 있습니다.

탭 또는 네비게이션 바와 같이 스크롤 가능한 FlowComponent에 고정시키려면 header 및 footer 속성을 사용할 수 있습니다.


# 네비게이션 바를 상단에 고정하기 
flow.header = flowBar
 
#탭 하단에 고정하기
flow.footer = tabBar
