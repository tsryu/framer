프레이머의 코드 탭에서는 코드 또는 디자인 변경점을 실시간으로 반영하는 미리보기 창이 표시됩니다. 이 창에서 직접 레이어를 드래그 앤 드롭하거나 크기를 조정할 수도 있습니다.

프레이머의 미리보기 창은 디자인 또는 코드 탭에서 만든 디자인을 실시간으로 반영합니다. 애니메이션을 다시 재생하려면 CMD + R을 누르세요. 창은 크롬으로 스냅 되지만 더 큰 작업 공간을 위해 완전히 분리될 수 있습니다. 여러 화면에서 디자인을 보려면 미리보기 창 상단 바에 있는 드롭다운 메뉴에서 선택해주세요.

디자인 탭에서 캔버스에 직접 디자인한 경우, 삽입한 모든 레이어는 상위 아트보드를 포함하지 않습니다. 레이어를 x : 0 및 y : 0 위칫값을 설정하면 미리보기 창의 왼쪽 위에 배치됩니다.
