프레이머의 코드 모드에서는 코드 또는 디자인에서 변경한 사항에 대한 실시간 시각적 피드백을 제공하는 미리보기 창이 표시됩니다. 이 창에서 직접 레이어를 드래그 앤 드롭하거나 크기를 조정할 수도 있습니다.

프레이머의 미리보기 창은 디자인 또는 코드 탭에서 만든 디자인에 대한 실시간 피드백을 제공합니다. 애니메이션을 다시 재생하려면 CMD + R을 누르십시오. 창은 크롬으로 스냅 되지만 더 큰 작업 공간에서 완전히 분리될 수 있습니다. 여러 화면에서 디자인을 보려면 미리 보기 창의 상단 네비게이션 바에 있는 드롭다운 메뉴에서 선택하기만 하면 됩니다.

디자인 모드에서 캔버스에 직접 디자인을 만든 경우, 삽입한 모든 레이어는 상위 아트보드를 포함하지 않습니다. 레이어를 x : 0 및 y : 0 위치값을 설정하면 미리보기 창의 왼쪽 위에 배치됩니다.
